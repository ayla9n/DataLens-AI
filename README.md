# 📈 DataLens AI

DataLens AI is a lightweight data insights tool built for small businesses. Upload sales, marketing, or finance data and get instant AI-powered insights.

---

## Features

- Automatically cleans data — handles missing values, duplicates, and column types
- Generates simple insights generated the dataset
- Recommends and renders the best charts
- Download cleaned data as a CSV

---

## Tech Stack

- **Streamlit** — dashboard
- **Groq API (LLaMA 3.3)** — AI insights and chart recommendations
- **Pandas** — data cleaning
- **Plotly** — charts

---

## Try It With Sample Data

Not sure where to start? The app comes with three **Sam's Bakery** sample datasets — just click to load, no upload needed.
